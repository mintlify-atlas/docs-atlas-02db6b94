# Welcome to Log

Log is a self-hosted platform that combines the best of development logging and memory keeping. Inspired by the need for a personal space to document development progress and life moments, Log gives you full control over your data while providing a beautiful, modern interface.

## Why Log?

After Hack Club's neighborhood ended, many developers felt the need for a platform to log their development process and share updates. Log fills that gap by providing:

- **Full ownership** of your data with self-hosted infrastructure
- **Flexible content types** - text posts, photos, and videos
- **Privacy controls** - draft, private, unlisted, and public posts
- **Collections** - organize related posts (trips, projects, etc.)
- **Tag system** - categorize and search your content
- **S3-compatible storage** - use any S3 provider for media files
- **Built-in authentication** - secure user management with Better-Auth

## Key Features

### Multiple Content Types

Create different types of posts to capture various moments:

- **Text posts** - Share thoughts, updates, and development logs
- **Photo posts** - Upload and share images (JPEG, PNG, GIF, WebP)
- **Video posts** - Share video content (MP4, WebM, MOV)

Each post type supports tags, collections, and customizable text descriptions.

### Privacy & Publishing Options

Control who sees your content with four visibility levels:

  <Card title="Draft" icon="file">
    Work in progress - only visible to you
  </Card>
  <Card title="Private" icon="lock">
    Published but only accessible when logged in

  <Card title="Unlisted" icon="link">

  <Card title="Public" icon="globe">

  </Card>

### Collections

Organize related posts into collections. Perfect for:
- Trip documentation
- Project development logs
- Themed content series
- Personal milestones

### Advanced Search

Log includes full-text search powered by PostgreSQL's text search capabilities:
- Search across all your posts
- Filter by tags
- Filter by content type
- Find posts by user

### URL Shortening

Built-in URL shortening service using `/u/[slug]` paths, or bring your own Shlink server.

## Technology Stack

Log is built with modern, reliable technologies:

- **Next.js 16** - React framework with App Router
- **Better-Auth** - Secure authentication with email/password
- **Drizzle ORM** - Type-safe database queries
- **PostgreSQL** - Robust relational database
- **Bun** - Fast JavaScript runtime and package manager
- **shadcn/ui** - Beautiful, accessible UI components
- **Tailwind CSS** - Utility-first styling
- **Sonner** - Toast notifications
- **Docker** - Containerized deployment

<Note>
  Log uses a monorepo structure with Bun workspaces, separating the web app, database layer, and authentication package for better maintainability.
</Note>
